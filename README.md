# Automated excel with Python 3

### Steps to use:

**Make sure you are inside _`/src`_ folder before running the command**

1. run `pip3 install -r requirements.txt`
2. run `cd ./src`
3. run `python3 index.py`

### Pylint score - 9.52

**Make sure you are inside _`/src`_ folder before running the command**

- run `pylint index.py`

### Pytest

**Make sure you are inside _`/src`_ folder before running the command**

- run `pytest tests.py`
